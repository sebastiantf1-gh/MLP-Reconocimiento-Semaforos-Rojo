# **MLP-Reconocimiento-Semaforos-Rojo**
## **Clasificador de Semáforos en Rojo**
Proyecto desarrollado para la asignatura de Deep Learning. Implementa un modelo MLP que alcanza un 98% de accuracy y un 99% de recall en la detección de luces rojas.

Se debe subir el dataset traffic_lights_dataset.zip a la carpeta /content del entorno de Google Colab.

Para ejecutar el paso 7 (pruebas), se proveen imágenes en la carpeta comprimida Imagenes_Test_Paso_7.zip
